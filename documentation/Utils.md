# Пакет utils

Пакет **`utils`** содержит вспомогательные инструменты, специфичные для работы с Selenium WebDriver.
Основная задача пакета — адаптация абстрактных структур данных из `core_lib` в форматы, понятные Selenium.

---

## 1. Класс `LocatorReader`

Утилитарный класс (Utility Class) для конвертации локаторов.
Он решает проблему несовместимости типов: фреймворк оперирует универсальным классом `locator.Locator`, а Selenium
требует свой собственный класс `org.openqa.selenium.By`.

### Методы

| Метод      | Параметры         | Описание и Логика                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
|:-----------|:------------------|:-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **`toBy`** | `Locator locator` | **Главный конвертер.**<br>Преобразует `Locator` в `By`.<br><br>**Алгоритм:**<br>1. Извлекает тип (`LocatorType`) и значение из входящего объекта.<br>2. Использует `switch expression` для выбора стратегии поиска Selenium:<br>    * `ID` -> `By.id()`<br>    * `XPATH` -> `By.xpath()`<br>    * `CSS` -> `By.cssSelector()`<br>    * и т.д.<br>3. **Валидация:** Если передан тип локатора, который Selenium не поддерживает (например, мобильные `ACCESSIBILITY_ID` или `IOS_PREDICATE`), метод выбрасывает `IllegalArgumentException`. Это предотвращает попытки использовать мобильные локаторы в веб-тестах. |
